What is Polkadot (DOT)?

Polkadot (DOT) is a multi-chain protocol that enables different blockchains to transfer messages and value in a trust-free way. Created by Ethereum co-founder Gavin Wood, it uses parachains to achieve scalability and interoperability. DOT is used for staking, governance, and bonding new parachains.
